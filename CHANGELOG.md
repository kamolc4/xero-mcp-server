# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows semantic versioning.

## [1.0.0] - 2026-07-04

### Added
- Production-ready TypeScript MCP server for Xero.
- Xero OAuth 2.0 flow with refresh-token support.
- MCP HTTP endpoint protected by API key authentication.
- Rate limiting for MCP requests.
- Health endpoint for operational checks.
- Xero MCP tools for invoices, contacts, accounts, and payments.
- Jest tests for config, health, authentication guard, and rate limiting.
- GitHub Actions CI workflow for lint, typecheck, tests, and build.
- MCPForge verification links and README badge.
